Catálogo Web con Carrito (Servlets + JSP)

Aplicación web desarrollada en Java utilizando Servlets y JSP que permite gestionar un catálogo de productos con carrito de compras en sesión.

Funcionalidades
Visualizar catálogo
Buscar productos
Filtrar por categoría
Agregar al carrito
Incrementar cantidad
Ver carrito
Calcular total
Limpiar carrito
Ejecución
mvn clean package
Copiar catalogo-web.war a Tomcat
Ejecutar Tomcat
Abrir: "http://localhost:8080/catalogo-web/catalogo"
